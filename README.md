# Car Phsyics Demo
A showcase of an arcade-y car physics system made in **Unity 6.4**.

Fully free to use (including 3d assets) under the **MIT License**. 

A working demo is available at https://musiu.itch.io/carphysicsdemo

-------

This project was fully developed by me, **MusiuDev** / **Ismael Loreto**.

3D graphics were made using Blender 5.1

The project doesn't include any sound effects as they were out of scope for this demo, but I may add some in the future.

**No AI tools were used at any point during the development of this project.**
